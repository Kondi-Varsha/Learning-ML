# My ML Learning Journey 🚀

## About
This repository documents my hands-on journey of learning 
AI/ML from the ground up — focusing on understanding concepts 
deeply through practice rather than just watching tutorials.

> "Don't just watch — experiment, struggle, and figure it out."

---

## My Approach
- Learn one concept at a time
- Immediately experiment with it hands-on
- Work on real datasets — not toy examples
- Check documentation when stuck — not tutorials
- Ask the right questions:
  *"What am I trying to do? What should I use? Why?"*

The biggest lesson wasn't pandas or numpy.

- Try first, take help later
