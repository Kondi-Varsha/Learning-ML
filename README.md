## 🗂️ Projects So Far

### 🏠 Airbnb NYC Data Cleaning & EDA

- Cleaned a real-world messy dataset with 26 columns & 102K rows
- Handled missing values, outliers, typos, and data type issues
- Performed full EDA — pricing, popularity, ratings, cancellation policy
- Visualised insights using Matplotlib & Seaborn
- First time working on a real dataset completely independently

**Key Findings:**
- Price difference across NYC neighbourhoods is only $7 — room type matters more than location
- Entire Home/Apt is most expensive but also most booked — people pay for privacy
- Flexible cancellation attracts more bookings but doesn't improve ratings
- Staten Island & Bronx are least popular regardless of price
