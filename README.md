# 🏙️ NYC Airbnb EDA

## 📌 About
Exploratory Data Analysis on NYC Airbnb dataset from Kaggle.
This project analyzes listing prices, room types, neighbourhood popularity,
host performance, and cancellation policies to understand customer preferences.

## ❓ Questions Answered
- Which neighbourhood group has the highest average price?
- Which neighbourhood group has the highest average rating?
- Which host has the most highly rated listings?
- Does cancellation policy affect bookings and review rates?
- Which room type do customers prefer?
- Which neighbourhood group is least popular?
- Is there a relationship between price and number of reviews?
- Does construction year affect listing price?

## 🔍 Key Findings
- All neighbourhood groups have very similar prices (difference of only $7)
- Queens is slightly the most expensive, Brooklyn the most affordable
- Entire Home/Apt is the most expensive and most booked room type
- Flexible cancellation policy attracts more bookings but doesn't improve ratings
- Staten Island and Bronx are the least popular neighbourhoods
- No relationship between price and number of reviews
- Building age has no effect on listing price

## 🎯 Conclusion
The ideal NYC Airbnb listing for maximum bookings would be an **Entire Home 
or Private Room** in **Manhattan or Brooklyn**, priced between **$50-$300**, 
with a **flexible cancellation policy**.
