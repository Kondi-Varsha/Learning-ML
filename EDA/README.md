# 🗽 NYC Airbnb Exploratory Data Analysis

## Project Overview

This project performs an exploratory data analysis (EDA) on NYC Airbnb listings to uncover insights about pricing, popularity, host performance, and booking behaviour across different neighbourhoods and room types.

---

## 📌 Questions Explored

- What is the average price by neighbourhood group?
- Which neighbourhood group has the highest average rating?
- Which neighbourhood is the most expensive?
- Which host has the most highly rated listings?
- Does a flexible cancellation policy affect review rate?
- What is the average price by room type?
- Which neighbourhood group is least popular?
- Which room type is least popular?

---

## 🔑 Key Findings

| Factor | Finding |
|--------|---------|
| 📍 Location | Price difference across neighbourhoods is only $7 — location barely matters |
| 🏠 Room Type | Entire Home/Apt is priciest but also most booked — privacy wins |
| 💰 Price | Most reviewed listings are priced between $50–$300 |
| 📋 Cancellation | Flexible policy attracts more bookings but doesn't improve ratings |
| ⭐ Rating | All neighbourhoods score similarly (3.0–3.5) |
| 🏗️ Building Age | Construction year has no effect on price or popularity |

---

## ✅ Conclusion

A customer looking for the best NYC Airbnb experience should choose an **Entire Home or Private Room** in **Manhattan or Brooklyn**, priced **under $300/night**, with a **flexible cancellation policy**.
